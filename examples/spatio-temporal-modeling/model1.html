Submodel 1: Basic Spatial Model

This submodel demonstrates a basic spatial prediction model using a DUMMY model.

R Code


# Load package
library(INLA)

# Simulate spatial data
coords <- expand.grid(x = 1:10, y = 1:10)
coords$z <- rnorm(100, mean = 5, sd = 2)

# Fit a spatial model
result <- inla(z ~ 1 + f(x, model = "rw2") + f(y, model = "rw2"),
               data = coords, family = "gaussian")

# Predict at new locations
pred <- predict(result, newdata = coords)
